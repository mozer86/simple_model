SIMPLE Simulation Input Output reference

Space

Represents a space within a building. This will often be a room, but it might also be half a room

Space {
   name : string,
   volume : number, // optional,
   infiltration : Infiltration, // optional,
   building : Building, // optional,
}

name

The name of the space

volume (optional)

Volume of the space

infiltration (optional)

The infiltration in the space

building (optional)

The importance of this space over time

API Access

// by name
let my_space = space(string);
// by index
let my_space = space(int);
