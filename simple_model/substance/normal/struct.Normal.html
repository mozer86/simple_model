Expand description

Represents a Normal; that is to say, a physical materiality with physical properties. The name Normal has been chosen instead of Material to respect EnergyPlus’ and other software’s terminology (which does not include Substace, but it does include Material, which is essentially a Normal with a thickness).

Fields

name: String

The name of the Normal. Should be unique for each Material in the SimpleModel object

index: Option<usize>

The position of this object in its contaner Vector

thermal_conductivity: Option<f64>

The thermal conductivity of the substance in W/m.K

specific_heat_capacity: Option<f64>

The specific heat capacity of the substance in J/kg.K

density: Option<f64>

The density of the substance in kg/m3

solar_absorbtance: Option<f64>

Solar absorbtance (from 0 to 1)

thermal_absorbtance: Option<f64>

Thermal absorbtance (i.e., emissitivy; from 0 to 1)
